No Plugins

Just like any other webpage, Shuteye only requires only a web browser and a URL. Currently Shuteye is supported by Chrome, Opera, and Firefox Beta.

Private

Shuteye uses the WebRTC standard to communicate encrypted video between browsers. Only the initial connection data is sent through a central server.
